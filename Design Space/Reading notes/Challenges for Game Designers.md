
Player Focused
- What is this game about
- how doi play
- how do i win?
- why do i want to play?
- what things do i need to do?

Players need a reason and desire to play the game.

Players need choices that have meaning

### Types of Design
- World Design
- System Design
- Content Design
- Game Writing
- Level Design
- User Interface

### Core of the game
Core statement such as
- This game is about....
- This game is the experience of being...
- This game teaches...
- This game simulates the experience of...

#### Core Game design Archetypes
- Territorial Acquisition - Risk, GO, King of the Hill
- Prediction - Rock, Paper Scissors; Roulette
- Spatial Reasoning - Tic-Tac-Toe, Connect 4
- Survival - Any survival game
- Destruction - FPS games
- Building - SimCity, Catan
- Collection - DK64, Sonic, Mario
- Chasing/Evading - Pac-man
- Trading - Pokémon, Catan
- Race - Speed running, Chutes and Ladders
